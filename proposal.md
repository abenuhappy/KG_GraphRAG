# GraphRAG 지식 그래프 시각화 Proposal

> 최초 작성: 2026-04-28 | 최종 수정: 2026-04-28 (3차)
> 문서 출처: 코드 역분석

---

## 1. 배경 및 목적

GraphRAG(Graph Retrieval-Augmented Generation)의 핵심 개념인 **계층형 지식 그래프 구조와 멀티레이어 탐색 흐름**을 시각적으로 설명하기 위한 HTML 다이어그램 문서입니다.

- 벡터 검색 단독 RAG의 한계(관계 추론 불가)를 보완하는 GraphRAG의 원리를 PM·기획자·비개발자도 이해할 수 있도록 표현합니다.
- 여행 도메인(일본 여행 추천)을 예시로, NER 추출 → 그래프 엔티티 매핑 → 멀티레이어 탐색 흐름을 한 화면에서 보여줍니다.

---

## 2. 주요 기능

- [x] 4개 레이어 지식 그래프 SVG 다이어그램
  - L1 지역 계층 (일본 → 교토·하코네·아라시야마)
  - L2 상품 (Package·Hotel·Ticket)
  - L3 스타일·속성 (Style·Keyword·Companion)
  - L4 사용자 검색 + NER 추출
- [x] 스키마 엣지 시각화 (HAS_CITY, LOCATED_IN, HAS_STYLE, RELATED_TO, SEARCHED)
- [x] GraphRAG 검색 경로 애니메이션 (청록색 점선 흐름)
- [x] `prefers-reduced-motion` 접근성 지원
- [x] SVG role/aria-labelledby 접근성 마크업
- [x] 한국어 설명 텍스트 및 피주석(figcaption)
- [x] **슬라이드 형식 설명 문서** (`index.html`, 구 `graphrag-slides.html`)
  - 슬라이드 01: 지식그래프의 트리플 구조 (주어·술어·목적어, 가중치·행동 로그)
  - 슬라이드 02: ES 색인 데이터 기준 4계층 구축 아이디어 (L1~L4 역할 및 활용 방식)
  - 슬라이드 03: GraphRAG 전체 흐름 (NER → 그래프 탐색 → LLM 생성)
  - 각 슬라이드 타이틀 하단에 한국어 텍스트 설명 추가 (`.slide-desc` 영역)
- [x] **GitHub 업로드 및 Vercel 배포 대응** — `index.html`로 rename 후 push
- [ ] 다른 도메인(커머스·콘텐츠 등) 예시 다이어그램 추가
- [ ] 인터랙티브 노드 클릭 시 설명 팝오버 표시
- [ ] 라이트/다크 모드 지원
